# 🎓 Student Result Processing System (C)

A **menu-driven result processing system** built using the C programming language.
This project allows efficient management and processing of student academic records.

---

## 🚀 Features

* 📥 Store Student Data
* 📊 Generate Report Cards
* 🏆 Display Rank List
* 🔍 Search Individual Student Records
* ❌ Exit System

## ⚙️ How It Works

The program runs using a **menu-driven interface**, where users can select options to perform different operations:

1. Enter student details (marks, roll number, etc.)
2. Generate and display report cards
3. Calculate and display rank list based on performance
4. Search for a specific student using roll number/name
5. Exit the program
